Xray_affection это проект для  для изучения физических явлений взаимодействия рентгена с объектами (фотоэффект, поглощение излучения, Комптоновское рассеяние). 
Рентгеновское излучение моделируется как монохроматическая волна гамма излучения
Объектом является металлическая пластина

Параметры:
src/global_parameters.cc
2 строка - изменение размера пикселя детектора (по умолчанию 10 мкм)
4 строка - изменение размера сетки (по умолчанию 100 пикселей)

При изменении размера пикселя автоматически меняются:
- Область стрельбы в PrimaryGenerator
- Размер детектора (кремниевого)
- Размер сцинтиллятора CsI
- Размер оптического клея
- Размер свинцовой пластины

src/PMPrimaryGenerator.cc
30 строка - изменение энергии излучения

macros/run.mac
5 строка - изменение количества моделируемых частиц

Порядок запуска:
Запуск из папки project/build
cmd:
cmake ..
make

Для визуализации
cmd:
./sim
Внутри Gui:
/run/beamOn N       # где N - число моделируемых событий

Для посчёта количества пройденных частиц
cmd:
./sim run.mac >out.txt
python3 count.py
